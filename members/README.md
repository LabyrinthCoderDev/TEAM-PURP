# New Member Template

Copy this folder to `members/[your-github-handle]/` and fill in:

1. `PROFILE.md` — who you are and what you build
2. `REPOS.md` — your active repositories
3. This `README.md` — optional, make it yours or delete it

Then:
- Add yourself to root `MEMBERS.md`
- Add yourself to root `README.md` (copy a member card)
- Add an entry to `JOURNAL.md`
- Open a pull request

See [CONTRIBUTING.md](../../CONTRIBUTING.md) for full instructions.
